# Pokedex
**Pokedex to see different pokemons and check their stats.Made with Html, Css and JavaScript. Using jQuery, Bootstrap for styles and Cypress and Jest for testing.**

### Installation:
- download or clone repo
- run: ***npm install***

#### to run Cypress:
- Install http-server

- run: ***npm run test:ui***

#### to run Jest:
- run: ***npm run test:dev***
